# measure-and-audit

## Goal

Combinar fotos con evidencia estructural, runtime y de cleanup.

## Instructions

Registrar revisión visual de fuente, conteos, bounds, VisualRevision, hitbox, estados y atributos relevantes. Verificar Edit, Client y Server por separado cuando exista comportamiento runtime. Tratar response.isError=true como error aunque el proceso externo termine con código 0. Confirmar que el set temporal desapareció y que no dejó scripts, labels, efectos o clones residuales.

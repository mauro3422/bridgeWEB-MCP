# define-contract

## Goal

Definir el artefacto canónico, baseline, aspectos evaluados, fases semánticas, estados y criterios Pass/Fail.

## Instructions

Clasificar el objetivo como modelo, crecimiento procedural, VFX, prop ambiental, UI del mundo, UI 3D o visual interactivo. Identificar la fuente canónica y preservar separadamente toda baseline aprobada. Definir fases semánticas, estados, escalas, distancias, cleanup, integración y evidencia técnica. Una candidata nunca reemplaza silenciosamente una baseline.

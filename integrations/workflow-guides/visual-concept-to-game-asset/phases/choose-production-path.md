# choose-production-path

## Goal

Elegir Roblox directo, Blender o un flujo híbrido según la geometría requerida.

## Instructions

Usar Roblox directo para primitivas, CSG simple, Beams, attachments y formas estilizadas de bajo detalle. Usar Blender cuando la silueta dependa de superficies continuas, deformación, gills, raíces fusionadas, UVs o normales. Evitar forzar Parts cuando producirán una corona de bolas o cilindros evidentes.

# build-parametric-prototype

## Goal

Construir una fuente canónica reproducible desde la receta, no un montaje manual irrepetible.

## Instructions

Crear parámetros nombrados y funciones por masa visual. Separar geometría, material, VFX, hitbox y estados. Conservar la receta junto con la fuente. Preparar un set aislado y otro integrado sin alterar autoridad de gameplay.

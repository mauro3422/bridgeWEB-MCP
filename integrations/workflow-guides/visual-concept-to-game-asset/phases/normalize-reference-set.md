# normalize-reference-set

## Goal

Convertir la dirección elegida en referencias útiles para modelado.

## Instructions

Obtener héroe, frontal, perfil, superior, tres cuartos, escala y fases/estados cuando apliquen. Mantener proporciones coherentes entre vistas; marcar inconsistencias del generador y decidir una vista maestra. Guardar originales sin sobrescribir.

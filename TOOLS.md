## Notas de seguridad

- No leer ni imprimir secretos, tokens ni archivos `.env`.
- No commitear `node_modules`, `dist`, logs, bases SQLite, sandbox ni binarios del tunnel-client.
- Para restart, usar `bridge_request_restart`; no matar procesos directamente desde el MCP activo.
- `run_command` es fallback, no el camino principal para editar o diagnosticar cuando hay tools especificas.
